# NestJS GraphQL Example

NestJS GraphQL Example with Apollo Server, PostgreSQL and Minio

## How to use
1. ```npm ci```
2. ```docker-compose up```
3. ```npm run dbmate up```
4. ```npm run db:seed```
3. ```npm run dev```
